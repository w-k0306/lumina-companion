# Lumina

> _Where silence finds its voice._

Lumina 是一个轻量级 AI 陪伴系统。不追求全能，只追求**在场感**。

## 设计哲学

- 🌙 **极简** — 没有多余的 UI 元素，只有你和对话
- 🫧 **呼吸感** — 界面会随时间和情绪微妙变化
- 🔮 **记忆** — 它记得你说过的事，但不会让你觉得被监视
- 🎐 **克制** — 不主动打扰，但永远在那里

## 技术栈

- **前端**: Vanilla JS + CSS Animations
- **后端**: Node.js / Python
- **AI**: OpenAI-compatible API
- **部署**: Vercel / Cloudflare Pages
